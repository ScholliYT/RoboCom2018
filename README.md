## Raspberry Pi
### Dependencies for python:
 numpy, imutils, opencv (cv2), pathlib, opencv-contib-python

pip (except opencv):

```$ pip install numpy imutils pathlib```

### Usage:
Start main program:

```$ python .\RPi\ball_tracking\ball_tracking.py```

Start configurator:

```$ python .\RPi\range-detector.py -f HSV -w```
